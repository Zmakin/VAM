# Dark Mode - Quick Reference

## Toggle Location
The dark mode toggle is located in the **top-right corner** of the Dashboard, next to the Settings (gear) icon.

## How to Use
1. **Click the slider** to toggle between dark and light modes
2. Your preference is **automatically saved**
3. The app will **remember your choice** next time you visit

## Visual Indicators
- **Moon icon** = Dark mode active (default)
- **Sun icon** = Light mode active
- **Blue slider** = Dark mode
- **Gray slider** = Light mode

## Default Mode
The app starts in **dark mode** by default for a comfortable viewing experience.

## Keyboard Shortcut
- Press **Tab** to focus on the toggle
- Press **Enter** or **Space** to switch modes

## What Changes in Dark Mode

### Colors
- Background: Dark gray instead of light gray
- Cards: Dark gray instead of white
- Text: White/light gray instead of dark gray
- Borders: Darker tones for subtle separation

## Troubleshooting

### Toggle Not Visible?
- Check top-right corner next to the gear icon
- Look for a small slider switch

### Mode Not Saving?
- Ensure browser allows localStorage
- Check browser is not in private/incognito mode
- Clear cache and reload page

### Colors Look Wrong?
- Try toggling the switch off and on
- Hard refresh the page (Ctrl+Shift+R or Cmd+Shift+R)
- Check for browser extensions that might interfere

## Technical Notes

### For Developers
- Theme state managed by React Context
- Preference stored in localStorage
- Tailwind CSS class-based dark mode
- Smooth transitions using CSS
